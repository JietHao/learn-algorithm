## 算法复杂度
### 时间复杂度（Time Complexity）
1. 常用时间复杂度
  - O(1): 常数
  - O(logN): 对数复杂度
  - O(n): 线性
  - O(n^2): 平方
  - O(n^3): 立方
  - O(2^n): 指数
  - O(n!): 阶乘
